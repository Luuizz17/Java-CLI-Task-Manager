# Java-CLI-Task-Manager
Um gerenciador simples de tarefas feito em Java para rodar no terminal.
O projeto foi criado para praticar Programação Orientada a Objetos (POO), uso de ArrayList, loops, e interação com o usuário via Scanner.

## Funcionalidades

•Criar tarefas

•Listar tarefas

•Marcar se uma tarefa está concluída

•Menu interativo no terminal

•Numeração automática das tarefas

•Tecnologias utilizadas

•Java

•Programação Orientada a Objetos (POO)

•ArrayList

## Como executar o projeto

1. Clone o repositório:

 git clone https://github.com/Luuizz17/java-cli-task-manager.git

2. Entre na pasta do projeto:

cd java-cli-task-manager

3. Compile o programa:

javac Tarefa.java

4.Execute o programa:

java Tarefa
